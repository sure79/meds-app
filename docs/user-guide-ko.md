## 1. 시작하기

### 1.1 프로젝트 생성

MEDS를 실행한 후, 새 프로젝트를 생성하는 방법입니다.

1. 메인 화면에서 **"새 프로젝트"** 버튼을 클릭합니다.
2. 프로젝트 기본 정보를 입력합니다:
   - **프로젝트명**: 선박명 또는 호선번호 (예: "Hull No. 3001")
   - **선주**: 선주명
   - **조선소**: 조선소명
   - **선급**: KR, ABS, DNV, LR, BV, NK, CCS 중 선택
   - **선종**: 벌크선, 컨테이너선, 유조선, 여객선 등
3. 전기 계통 기본 사양을 설정합니다:
   - **주 전압**: 220V, 380V, 440V, 450V, 480V, 690V, 720V, 6600V, 11000V
   - **주파수**: 50Hz 또는 60Hz
   - **상수**: 3상
   - **접지 방식**: IT, TN-S, TN-C, TN-C-S, TT
4. **"생성"** 버튼을 클릭하여 프로젝트를 완료합니다.

### 1.2 프로젝트 열기

- 메인 화면의 **"최근 프로젝트"** 목록에서 선택하거나
- **"프로젝트 열기"**를 클릭하여 저장된 프로젝트 파일(.meds)을 불러옵니다.

### 1.3 화면 구성

MEDS의 기본 화면은 다음과 같이 구성됩니다:

- **상단 네비게이션**: 모듈 전환 탭 (전력균형, 단선결선도, 단락전류, 전압강하, 보호협조, 선급제출)
- **좌측 패널**: 프로젝트 트리 및 부하 목록
- **중앙 작업영역**: 현재 모듈의 주 작업 공간
- **우측 패널**: 속성 편집기 (선택한 항목의 상세 정보)
- **하단 상태바**: 프로젝트 정보 및 계산 상태

## 2. 전력균형 (Load Balance)

전력균형은 선박 전기 설계의 첫 단계로, 각 운전 조건에서의 전력 수요를 산출합니다.

### 2.1 부하 입력

#### 부하 추가 방법
1. 좌측 패널에서 **"부하 추가"** 버튼을 클릭합니다.
2. 부하 정보를 입력합니다:
   - **부하명**: 장비 이름 (예: "M/E L.O. Pump No.1")
   - **부하 ID**: 고유 식별 번호 (예: "EP-101")
   - **정격 출력 (kW)**: 장비의 정격 용량
   - **전압 (V)**: 장비에 공급되는 전압
   - **역률 (PF)**: 장비의 역률 (0.1 ~ 1.0)
   - **효율 (%)**: 장비 효율 (일반적으로 85~95%)
   - **기동 방법**: DOL / Y-Delta / Soft Starter / VFD / None
   - **부하 그룹**: 기관실, 갑판, 거주구, 비상 등

#### 프리셋 부하 활용
- **"프리셋 불러오기"** 기능으로 일반적인 선박 장비 목록을 빠르게 추가할 수 있습니다.
- 선종별 표준 부하 목록(벌크선, 컨테이너선, 유조선 등)이 제공됩니다.

#### 부하 일괄 입력
- Excel 파일(.xlsx)에서 부하 목록을 가져올 수 있습니다.
- **"가져오기"** 버튼 클릭 후 Excel 파일을 선택합니다.
- 템플릿 파일은 **"템플릿 다운로드"**로 받을 수 있습니다.

### 2.2 운전 조건 설정

각 부하에 대해 운전 조건별 부하율(Load Factor)과 가동율(Duty Factor)을 설정합니다.

| 운전 조건 | 설명 |
|----------|------|
| 항해 (Sea Going) | 정상 항해 상태 |
| 입출항 (Maneuvering) | 항만 접안/이안 |
| 정박-하역 (Port Loading) | 항만에서 하역 작업 중 |
| 정박-무하역 (Port Idle) | 항만에서 대기 |
| 비상 (Emergency) | 비상 전원 운전 |
| 묘박 (Anchor) | 투묘 대기 |
| DP 운전 (Dynamic Positioning) | DP 모드 운전 (해당 시) |
| 특수조건 1 | 사용자 정의 조건 |
| 특수조건 2 | 사용자 정의 조건 |

각 조건에서:
- **부하율 (K_load)**: 정격 대비 실제 사용 비율 (0~1)
- **가동율 (K_duty)**: 해당 조건에서 장비 운전 확률 (0~1)

### 2.3 발전기 구성

1. **"발전기 설정"** 탭을 선택합니다.
2. 발전기 정보를 입력합니다:
   - 정격 용량 (kW)
   - 정격 전압 (V)
   - 정격 역률
   - 대수
   - 각 운전 조건별 가동 대수
3. 표준 발전기 크기(100, 150, 200, 250, 300, 400, 500, 600, 750, 800, 1000, 1500, 2000 kW)를 참조하여 선정합니다.

### 2.4 계산 및 결과 확인

**"계산 실행"** 버튼을 클릭하면 다음 결과가 표시됩니다:

- **총 유효전력 수요 (kW)**: 각 운전 조건별
- **총 무효전력 수요 (kVar)**: 각 운전 조건별
- **총 피상전력 수요 (kVA)**: 각 운전 조건별
- **종합 역률**: 각 운전 조건별
- **발전기 부하율 (%)**: 가동 중인 발전기 대비
- **선급 여유율 판정**: 적합/부적합 표시

결과는 표 형태와 막대 그래프로 동시에 표시됩니다.

### 2.5 자동 최적화

**"발전기 최적화"** 기능을 사용하면:
- 모든 운전 조건을 만족하는 최소 발전기 구성을 자동으로 제안합니다.
- 표준 발전기 크기를 기준으로 최적 조합을 찾습니다.

## 3. 단선결선도 (Single Line Diagram)

### 3.1 SLD 개요

단선결선도(SLD)는 전력 계통의 구성을 단선으로 표현한 도면입니다. MEDS에서는 직관적인 드래그 앤 드롭 인터페이스로 SLD를 작성할 수 있습니다.

### 3.2 SLD 작성

#### 기기 배치
1. 좌측 **기기 팔레트**에서 필요한 기기를 선택합니다:
   - **발전기 (Generator)**: 주 발전기, 비상 발전기, 축발전기
   - **버스바 (Busbar)**: 주 배전반 버스, 비상 배전반 버스
   - **차단기 (Circuit Breaker)**: ACB, MCCB
   - **변압기 (Transformer)**: 강압 변압기, 절연 변압기
   - **전동기 (Motor)**: 유도 전동기, 동기 전동기
   - **부하 (Load)**: 일반 부하, 조명, 히터
   - **기타**: 축전지(Battery), UPS, Shore Connection
2. 기기를 중앙 캔버스로 드래그하여 원하는 위치에 놓습니다.
3. 기기를 클릭하면 우측 패널에서 속성을 편집할 수 있습니다.

#### 연결선 그리기
1. 기기의 **접속점(터미널)**에 마우스를 가져가면 연결 포인트가 표시됩니다.
2. 접속점을 클릭한 후 연결할 대상 기기의 접속점까지 드래그합니다.
3. 연결선은 자동으로 정리되며, 수동으로 중간점을 추가할 수 있습니다.

#### 버스바 연결
- 발전기에서 버스바로, 버스바에서 부하측 차단기로 연결합니다.
- 버스바 간 연결(Bus-Tie)은 차단기를 통해 연결합니다.

### 3.3 SLD 편집 기능

- **확대/축소**: 마우스 스크롤 또는 +/- 버튼
- **이동**: 캔버스 빈 영역을 드래그
- **선택**: 클릭으로 단일 선택, Ctrl+클릭으로 다중 선택
- **정렬**: 선택한 기기들을 상/하/좌/우 정렬
- **실행취소/재실행**: Ctrl+Z / Ctrl+Y
- **복사/붙여넣기**: Ctrl+C / Ctrl+V
- **그룹**: Ctrl+G로 선택한 기기 그룹화

### 3.4 SLD와 다른 모듈의 연동

SLD에서 생성한 계통 구성은 다른 모듈에 자동으로 반영됩니다:
- 발전기와 부하 정보는 **전력균형** 모듈과 동기화
- 임피던스 정보는 **단락전류** 계산에 활용
- 케이블 정보는 **전압강하** 계산에 활용
- 보호장치 정보는 **보호협조** 해석에 활용

## 4. 단락전류 해석 (Short Circuit Analysis)

### 4.1 해석 개요

단락전류 해석은 계통의 각 지점에서 발생 가능한 단락전류를 계산하여 차단기 정격을 선정하고 보호 협조의 기초 데이터를 산출합니다.

### 4.2 입력 데이터

#### 발전기 데이터
- 정격 용량 (kVA 또는 kW + PF)
- 정격 전압 (V)
- 직축 차과도 리액턴스 X"d (pu) -- 일반적으로 0.10~0.20
- 직축 과도 리액턴스 X'd (pu) -- 일반적으로 0.15~0.30
- 역상 리액턴스 X2 (pu)
- 영상 리액턴스 X0 (pu)
- R/X 비율

#### 전동기 데이터
- 정격 용량 (kW)
- 차과도 리액턴스 X"d (pu) -- 일반적으로 0.16~0.25
- 기동 전류 배수

#### 케이블 데이터
- 단면적 (mm²)
- 길이 (m)
- 저항 R (Ω/km)
- 리액턴스 X (Ω/km)

#### 변압기 데이터 (해당 시)
- 정격 용량 (kVA)
- 임피던스 전압 (%)
- 권선 연결 방식

### 4.3 해석 실행

1. SLD에서 단락 지점을 선택합니다 (마우스 우클릭 > "단락점 지정").
2. 또는 **"단락점 추가"** 버튼으로 버스바 또는 특정 위치를 지정합니다.
3. **"해석 실행"**을 클릭합니다.

### 4.4 결과 확인

각 단락점에 대해 다음 결과가 표시됩니다:

| 항목 | 기호 | 단위 | 설명 |
|------|------|------|------|
| 초기 대칭 단락전류 | I"k | kA (rms) | 단락 직후의 대칭 전류 |
| 첨두 단락전류 | i_p | kA (peak) | 비대칭 첨두값 |
| 차단전류 | I_b | kA (rms) | 차단기 동작 시점의 전류 |
| 정상 단락전류 | I_k | kA (rms) | 충분한 시간 경과 후 전류 |

결과는 SLD 위에 직접 표시되며, 상세 보고서 형태로도 확인 가능합니다.

### 4.5 차단기 정격 선정

계산된 단락전류를 기반으로 차단기 정격을 자동 추천합니다:
- 표준 차단기 정격: 10, 15, 20, 25, 35, 50, 65, 85, 100, 150 kA
- 계산된 단락전류보다 큰 최소 표준 정격을 선정합니다.

## 5. 전압강하 해석 (Voltage Drop Analysis)

### 5.1 해석 개요

케이블 길이와 부하 전류에 따른 전압강하를 계산하여 선급 기준 이내인지 확인합니다.

### 5.2 케이블 데이터 입력

각 급전선(Feeder)에 대해 다음 정보를 입력합니다:

- **케이블 종류**: TPYC, TPYCY, DPYC, DPYCY, BPYCY 등
- **도체 재질**: 동(Cu) 또는 알루미늄(Al)
- **단면적**: 1.5 ~ 400 mm²
- **심수**: 2심, 3심, 4심
- **길이**: 케이블 포설 경로를 따른 편도 길이 (m)
- **포설 방법**: 트레이(Tray), 덕트(Duct), 단독(Single)
- **주위 온도**: 기관실(45°C), 갑판(40°C), 거주구(35°C)

### 5.3 계산 실행 및 결과

**"전압강하 계산"** 버튼 클릭 시 각 급전선에 대해 계산됩니다:

- **운전 시 전압강하 (%)**: 정상 운전 전류 기준
- **기동 시 전압강하 (%)**: 기동 전류 기준 (전동기 부하)
- **판정**: 선급 기준 대비 적합/부적합

부적합 항목은 빨간색으로 강조 표시됩니다.

### 5.4 케이블 자동 선정

**"케이블 자동 선정"** 기능을 사용하면:
1. 허용 전류 조건 만족
2. 전압강하 조건 만족
3. 단락 내량 조건 만족

세 가지 조건을 모두 만족하는 최소 케이블 단면적을 자동으로 선정합니다.

## 6. 보호협조 (Protection Coordination)

### 6.1 해석 개요

보호협조 해석은 보호장치(차단기, 퓨즈 등)의 동작 특성을 TCC(Time-Current Characteristic) 곡선 위에 표시하여 선택성(Selectivity)을 확인합니다.

### 6.2 보호장치 설정

#### ACB (기중 차단기) 설정
- 정격 전류 (A)
- Long Time 설정: 전류 / 시간
- Short Time 설정: 전류 / 시간
- Instantaneous 설정: 전류
- Ground Fault 설정: 전류 / 시간

#### MCCB (배선용 차단기) 설정
- 정격 전류 (A)
- 트립 특성: B / C / D
- 순시 트립 배수

### 6.3 TCC 곡선 작성

1. 보호협조를 검토할 경로(Path)를 선택합니다.
   - 예: 발전기 ACB > 주 배전반 버스 > 분기 MCCB > 전동기
2. 해당 경로의 모든 보호장치가 TCC 다이어그램에 표시됩니다.
3. 함께 표시되는 곡선:
   - 케이블 열적 한계 곡선 (Cable Damage Curve)
   - 전동기 기동 특성 곡선 (Motor Starting Curve)
   - 발전기 과부하 한계 곡선 (Generator Overload Curve)

### 6.4 선택성 확인

TCC 곡선에서 다음 사항을 확인합니다:

- **상위-하위 보호장치 간 간격**: 최소 0.1~0.3초 시간 여유
- **케이블 보호**: 차단기 곡선이 케이블 손상 곡선 아래에 위치
- **전동기 기동 간섭 없음**: 기동 곡선이 차단기 트립 곡선 아래에 위치
- **발전기 보호**: 과부하 곡선 이내에서 차단

부적합한 구간은 경고 메시지와 함께 해당 영역이 강조 표시됩니다.

### 6.5 설정값 최적화

**"자동 조정"** 기능을 사용하면 선택성을 확보하면서 보호장치 설정값을 자동으로 최적화합니다.

## 7. 선급 제출 (Class Submission)

### 7.1 제출 서류 생성

선급 제출에 필요한 서류를 자동으로 생성합니다:

| 서류 | 설명 | 형식 |
|------|------|------|
| 전력균형표 | Electrical Load Balance | Excel / PDF |
| 단선결선도 | Single Line Diagram | PDF / DWG |
| 단락전류 계산서 | Short Circuit Calculation | PDF |
| 전압강하 계산서 | Voltage Drop Calculation | PDF |
| 보호협조 해석서 | Protection Coordination Study | PDF |

### 7.2 선급별 양식 적용

- 선택한 선급에 따라 적절한 양식과 기준이 자동으로 적용됩니다.
- 선급 마크 및 프로젝트 정보가 서류 머리글에 자동 삽입됩니다.

### 7.3 검증 보고서

**"검증 실행"** 버튼을 클릭하면:
- 모든 계산 결과가 선급 기준에 적합한지 자동 검증
- 부적합 항목 목록 및 권장 조치 사항 제시
- 검증 통과 시 "선급 제출 준비 완료" 상태 표시

### 7.4 제출

1. 모든 검증을 통과한 후 **"제출 서류 생성"** 클릭
2. 서류 패키지(ZIP)가 생성됩니다.
3. 생성된 서류를 검토한 후 선급 포털에 업로드하거나 인쇄하여 제출합니다.

## 8. 내보내기 기능 (Export)

### 8.1 지원 형식

| 형식 | 용도 |
|------|------|
| PDF | 인쇄용 보고서 및 도면 |
| Excel (.xlsx) | 전력균형표, 케이블 스케줄 등 데이터 |
| JSON | 프로젝트 데이터 교환 |
| PNG / SVG | 다이어그램 이미지 |

### 8.2 내보내기 방법

1. 각 모듈에서 **"내보내기"** 버튼을 클릭합니다.
2. 내보내기 형식을 선택합니다.
3. 저장 위치를 지정하고 **"저장"**을 클릭합니다.

### 8.3 일괄 내보내기

**"전체 내보내기"** 기능으로 프로젝트의 모든 결과를 한 번에 내보낼 수 있습니다:
- 모든 계산서를 PDF로 생성
- 모든 데이터를 Excel로 생성
- ZIP 파일로 압축하여 저장

## 9. 유용한 팁 및 주의사항

### 9.1 일반 팁

- **자동 저장**: MEDS는 5분마다 프로젝트를 자동 저장합니다. 수동 저장은 Ctrl+S입니다.
- **실행취소**: 대부분의 작업은 Ctrl+Z로 취소할 수 있습니다 (최대 50단계).
- **단축키**: F1을 눌러 전체 단축키 목록을 확인하세요.
- **한/영 전환**: 부하명과 설명은 한글과 영문을 동시에 입력할 수 있습니다.
- **단위 체계**: 설정에서 SI 단위와 Imperial 단위를 전환할 수 있습니다.

### 9.2 전력균형 관련

- 부하율과 가동율을 정확히 구분하세요:
  - **부하율**: 장비가 운전 중일 때 정격 대비 실제 사용 비율
  - **가동율**: 해당 운전 조건에서 장비가 운전될 확률
- 비상 조건에서는 SOLAS에 따른 필수 비상 부하만 포함하세요.
- 발전기 부하율이 80%를 초과하면 경고가 표시됩니다.
- 발전기 부하율이 40% 미만이면 과잉 설비 경고가 표시됩니다.

### 9.3 단락전류 관련

- X"d 값은 발전기 제조사 데이터를 사용하세요. 값이 없는 경우 일반적인 기본값(0.15 pu)이 적용됩니다.
- 전동기 기여를 반드시 포함하세요. 전동기 기여를 무시하면 단락전류가 과소 평가됩니다.
- 차단기 정격은 계산된 단락전류보다 충분히 큰 값을 선정하세요.

### 9.4 전압강하 관련

- 케이블 길이는 실제 포설 경로를 따른 길이를 사용하세요 (직선 거리가 아님).
- 온도 보정을 반드시 적용하세요. 기관실은 45°C, 갑판은 40°C가 일반적입니다.
- 기동 시 전압강하가 15%를 초과하면 기동 방법 변경(DOL에서 Y-Delta 또는 VFD로)을 고려하세요.

### 9.5 보호협조 관련

- TCC 곡선에서 상위-하위 차단기 곡선이 교차하면 선택성이 확보되지 않은 것입니다.
- 발전기 ACB의 Long Time 설정은 발전기 과부하 내량 이내여야 합니다.
- 최종 MCCB의 순시 트립 설정은 전동기 기동 전류보다 높아야 합니다.

### 9.6 주의사항

- 계산 결과는 참고용이며, 최종 설계는 반드시 자격을 갖춘 전기 엔지니어가 검토해야 합니다.
- 선급 규정은 수시로 개정되므로, 최신 규정을 반드시 확인하세요.
- MEDS의 계산은 IEC 61363-1을 기반으로 하나, 특수한 경우(고압 계통, 특수 선종 등)에는 추가적인 해석이 필요할 수 있습니다.
- 프로젝트 파일(.meds)은 정기적으로 백업하시기 바랍니다.
- 비상 전원 계통의 설계는 SOLAS 요구사항을 최우선으로 준수해야 합니다.
